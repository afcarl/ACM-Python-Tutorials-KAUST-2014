Python Tutorials: Day 1 - Basics

This is a beginners' level session, in which we focus on the basic syntax of the language and essential functionality. Using ipython pre-prepared notebooks, thanks to Tareq Malas, we will follow a hands on approach of understanding the different syntax features and to go to other sites for further details on usage.

Python Tutorials: Day 2 - Intermediate topics

Numerics' level session

In this Numeric's level sesion we focus here on two special packages in python: Numpy and Matplotlib. These two libraries are especially useful for Numerical Linear Algebra and for plotting. Using ipython pre-prepared notebooks, thanks to Prof. Ketcheson and Yiannis Hadjimichael, we follow a hands on approach to understanding the different features and functionality of these packages and provide useful links for further details on usage.
