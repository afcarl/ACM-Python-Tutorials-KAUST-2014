The tutorials are logically split into 3 parts: Basic, Intermediate, and Advanced levels. Each tutorial level is given on a separate day. The morning sessions of the tutorials cover general Python topics of various levels; the afternoon sessions are devoted to specific techniques, packages, and problems that could be easily solved with Python. Finally, we have a 2-hour session on MPI with Python.

Day 1: Basics

This is a beginners' level session, in which we focus on the basic syntax of the language and essential functionality. The session is hand-on.

General topics: Getting started with Python: Using iPython notebook, basic usage and syntax

Special topics: Useful tools for automated script generation; terminal commands execution and parsing results; working with files: reading, parsing, using encodings, and writing to files.

Day 2: Intermediate topics

This session is focused on stepping beyond basics and also on some useful numerical packages used by scientists world wide: Numpy, SciPy, and Matplotlib. The session is hand-on.

General topics: Python libraries, programming approaches in python, advanced syntax usage, input and output.

Special topics: Scientific calculations with NumPy and SciPy, advanced plotting techniques with MatPlotLib.

Day 3: Advanced techniques

During this session we go further from procedural programming to object-oriented style and functional approaches widely used in professional Python programming. The session is hand-on.

General topics: Classes and objects, object-oriented programming, functional programming, memory management.

Special topics: Optimization in Python with OpenOpt; machine learning in Python with Scikit-Learn.

Day 4: MPI with Python

During this session you will learn about Message Passing Interface (MPI) programming approach in high performance computing and about how to work with it in Python. The session is lecture-like.

We hope that you will enjoy the tutorials and will learn skills you can apply in your daily research and programming. Looking forward to seeing you there!
